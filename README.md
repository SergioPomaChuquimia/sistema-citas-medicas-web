# Sistema Web de Administración de Citas Médicas

## ¿De qué trata este proyecto?

Este sistema fue desarrollado para el Hospital General con el objetivo de automatizar el proceso de consultas médicas. Antes de este sistema, las citas se gestionaban manualmente. Ahora los pacientes pueden registrarse, agendar citas con especialistas y el personal médico puede administrar horarios, especialidades y consultas desde un panel web.

## Tecnologías utilizadas

- **Backend:** Laravel (PHP)
- - **Frontend:** Angular
  - - **Base de datos:** MySQL
   
    - ## Funcionalidades principales
   
    - - Registro e inicio de sesión de pacientes
      - - Agendamiento de citas por especialidad
        - - Administración de médicos y horarios
          - - Gestión de consultas y historial de pacientes
            - - Panel de administración para el personal del hospital
             
              - ## Contexto
